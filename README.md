# Portfolio Tracker - Frontend

React frontend dla aplikacji do śledzenia portfela akcji.

## Demo

🌐 **Live:** https://portfolio-tracker-frontend-one.vercel.app

## Repozytoria

- **Frontend:** https://github.com/llukasz4/portfolio-tracker-frontend
- **Backend:** https://github.com/llukasz4/portfolio-tracker

## Stack technologiczny

- React
- TradingView Widget (wykresy)
- lightweight-charts (wykresy sekundowe)
- react-rnd (draggable panels)
- Axios

## Uruchomienie lokalne

```bash
npm install
npm start
```

Aplikacja działa na `http://localhost:3000`

## Zmienne środowiskowe

Stwórz plik `.env`:

```
REACT_APP_FINNHUB_KEY=twój_klucz_finnhub
```
